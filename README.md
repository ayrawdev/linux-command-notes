# Linux Commands for Cloud Engineers

This repository contains essential Linux commands commonly used by **Cloud Engineers, DevOps Engineers, and System Administrators**.

---

## File & Directory Management

```bash
ls            # list files and directories
ls -la        # list hidden files with detailed information
pwd           # show current directory
cd /var       # change directory
cp file1 file2   # copy file1 to file2
mv file1 file2   # move or rename file
rm file.txt      # delete a file
```

---

## File Permissions

```bash
chmod 755 file.sh   # change file permissions
chown user file.txt # change file ownership
```

---

## Disk Usage

```bash
df -h      # show disk space usage
du -sh     # show size of directory
```

---

## Process Management

```bash
ps aux      # show running processes
top         # real-time system monitoring
kill -9 PID # force kill process
```

---

## Networking Commands

```bash
ping google.com   # test network connectivity
ip a              # show network interfaces
ifconfig          # legacy command (older systems)
netstat -tulnp    # list open ports and services
```

---

## System Information

```bash
uname -a    # show system information
whoami      # show current user
uptime      # system uptime
hostname    # system hostname
```

---

## Package Management (Ubuntu)

```bash
sudo apt update
sudo apt upgrade
sudo apt install package_name
sudo apt remove package_name
```

---

## File Search

```bash
find / -name filename   # search for file
grep "word" file.txt    # search word in file
```

---

## Useful Commands for Cloud Engineers

```bash
ssh user@server-ip                    # connect to remote server
scp file.txt user@server-ip:/home/user # copy file to remote server
sudo systemctl status nginx           # check service status
sudo systemctl restart nginx          # restart service
```

---

## Log Monitoring

```bash
tail -f /var/log/syslog       # monitor logs in real time
less /var/log/auth.log        # view authentication logs
grep "failed" /var/log/auth.log # search failed login attempts
```
